# Phantom Wallet

> Solana wallet interface

## Features

- Send/Receive
- Token swaps
- NFT gallery
- Staking

## Tech Stack

- Frontend: HTML5, CSS3, JavaScript
- Blockchain: Solidity, Rust, Web3.js
- AI: MiMo V2 Flash, Claude Code

## Installation

```bash
git clone https://github.com/bimolasi/phantom-wallet.git
cd phantom-wallet
npm install
npm run dev
```
